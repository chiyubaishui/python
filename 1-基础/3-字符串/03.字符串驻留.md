先看下面这段代码
>>> a = 'apple'
>>> b = 'apple'
>>> a is b
True

再来看这段：
>>> a = 'apple!'
>>> b = 'apple!'
>>> a is b
False

第一段代码，a 和 b 的值都是 apple，当解释器准备创建第二个字符串”apple”时，解释器会去驻留内存中查找是否已经存在“apple”对象，如果有就不在创建新的对象了，是共享一个对象。

但并不是所有的字符串有会放进驻留内存中，如果都驻留的话，内存会爆炸。只有符合如下规则才有可能被驻留
长度为 0 或者 1 的字符串被驻留.
字符串中只包含字母，数字或下划线时将会驻留。
字符串必须是编译时的常量字符串
